# Group 22 Spam Email Analysis

This repository contains the exploratory data analysis for Group 22's spam email classification coursework.

## Files

- `Group_22_Analysis.qmd`: main Quarto source file
- `Group_22_Analysis.html`: rendered output
- `dataset22.csv`: dataset used in the analysis
- `DAS-Group-22.Rproj`: RStudio project file

## How to run

Open `DAS-Group-22.Rproj` in RStudio, then open `Group_22_Analysis.qmd` and render the document.

Required R packages:

- `tidyverse`
- `skimr`
- `janitor`
- `here`
- `GGally`

## Notes

- The analysis currently focuses on the exploratory data analysis section.
- The dataset contains duplicated rows, which are noted in the report and retained at this stage.
